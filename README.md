# terraform-alart-sample

## Usage

```sh
% terraform init
% terraform validate
% terraform fmt -recursive
% terraform plan -var 'security_group_ids=[ "xxxxxxxxxxx" ]' -var 'subnet_id=xxxxxxxxxxxxxxxxx'
% terraform plan -var 'security_group_ids=[ "xxxxxxxxxxx" ]' -var 'subnet_id=xxxxxxxxxxxxxxxxx'
% terraform destroy -var 'security_group_ids=[ "xxxxxxxxxxx" ]' -var 'subnet_id=xxxxxxxxxxxxxxxxx'
```
